12색상환 (Color Wheel)

인접한 색끼리는 부드럽고, 마주 보는 색끼리는 강한 대비를 만듭니다.

배색 방법 — 어울리는 색 고르기

보색 (Complementary)

색상환에서 정반대 위치에 있는 두 색.
강한 대비 — 강조 포인트에 효과적.

유사색 (Analogous)

이웃한 3색.
안정적이고 부드러운 분위기 — 단조롭지 않게 한 색을 살짝 강조하세요.

삼각 (Triadic)

색상환에서 120도 간격 3색.
활기차고 균형 잡힌 느낌 — 한 색은 메인, 나머지는 보조.

단색(Monochrome)

같은 색의 명도만 변화.
차분하고 통일감 — 비즈니스·미니멀 디자인에 적합.

시멘틱 컬러 (의미가 정해진 색)

웹·앱 UI에서 이미 의미가 굳어진 색입니다. 가급적 관행을 따라야 사용자가 헷갈리지 않습니다.

성공 · Success #10b981 완료, 확인, 저장됨
정보 · Info #3b82f6 안내, 링크, 도움말
주의 · Warning #f59e0b 경고, 확인 필요
오류 · Error #ef4444 실패, 삭제, 위험

색 고를 때 쓰는 도구

  • Coolors — 스페이스바로 무한히 새 팔레트 생성
  • Adobe Color — 색상환 기반 배색 (보색·유사색·삼각)
  • Tailwind Colors — 22개 컬러 × 11단계 (이 페이지 컬러 출처)
  • UI Colors — Tailwind 형식으로 9단계 자동 생성
  • Realtime Colors — 실제 페이지에 색을 입혀 미리보기
  • WebAIM Contrast Checker — 텍스트 대비비 검증 (가독성 필수)
  • Coblis — 색맹 시뮬레이터 (접근성 점검)
팁 — 색은 적게, 의미는 분명히
  • 한 화면에 메인 1 + 보조 1 + 강조 1 = 3색이면 충분합니다.
  • 중성색(회색)은 색 수에서 빼고 마음껏 쓰세요.
  • 색만으로 정보를 전달하지 마세요 — 색맹 사용자를 위해 아이콘·텍스트와 함께 씁니다.

프롬프트 모음 (강의에서 사용한 것들)

AI 기초 실습 - 자주 쓰는 프롬프트

# 요약
"다음 글을 3줄로 요약해줘. 대상 독자는 [누구]야. [어떤] 톤으로."

# 이메일
"나는 [신분]이고, [받는 사람]에게 [목적]에 대한 이메일을 [몇 자] 정도로 [톤] 써줘."

# 학습
"나는 [수준]이야. [개념]을 일상생활 비유 1개와 흔한 오해 1개를 포함해서 설명해줘."

# 비교
"[A]와 [B]를 [기준 5가지]로 비교한 표를 만들고, 상황별 추천을 알려줘."

# 브레인스토밍
"[상황]에 대한 아이디어 10개 필요해. 평범한 5개 + 엉뚱한 5개로 줘."

바이브 코딩 - 자주 쓰는 프롬프트

# 코드 설명 요청
"이 코드의 각 줄이 무슨 일을 하는지 한국어 주석으로 설명해줘.
비전공자가 가장 헷갈릴 부분 3개도 짚어줘."

# 에러 디버깅
"다음 코드를 실행했는데 에러가 났어:
[코드]
에러: [에러 메시지]
원하던 것: [의도]
원인과 해결 방법을 알려줘."

# 웹페이지 생성
"[누구]를 위한 [어떤] 웹페이지를 만들어줘.
포함 섹션: [목록]
디자인 톤: [톤]
반응형 필수. HTML+CSS 한 파일로."

# 기능 추가
"이 페이지에 [기능]을 추가해줘.
기존 구조는 건드리지 말고, 새 코드만 표시해줘."

자주 묻는 질문

Q. Claude Pro 구독은 꼭 필요한가요?

네.
무료 플랜은 사용 한도가 빠르게 차서 강의 실습에 부족합니다.
Pro($20/월) 또는 학생 단체 할인을 활용해 주세요.

Q. Cursor는 유료 플랜이 필요한가요?

아닙니다.
무료 플랜으로도 한 달에 충분한 양을 쓸 수 있고, 본인이 가진 Claude Pro 구독을 연결해서 쓰면 한도가 훨씬 늘어납니다.

Q. 컴맹인데 따라갈 수 있을까요?

네.
이 강의는 "비전공자 초보"가 기본 대상입니다.
AI 도구가 어려운 부분을 대신 해주기 때문에, 옛날 코딩 강의보다 진입 장벽이 훨씬 낮습니다.

Q. 결과물은 어떻게 활용할 수 있나요?

본인이 만든 페이지는 본인 소유입니다.
포트폴리오·이력서·SNS 어디든 자유롭게 활용 가능합니다.

Q. 강의 후에도 계속 배우려면?

강의 5일차 마지막에 본인 관심사에 따른 다음 단계 트랙을 안내합니다 (심화 웹개발 / AI API / 모바일 / 데이터 분석).
