프롬프트 모음 (강의에서 사용한 것들)

AI 기초 실습 - 자주 쓰는 프롬프트

# 요약
"다음 글을 3줄로 요약해줘. 대상 독자는 [누구]야. [어떤] 톤으로."

# 이메일
"나는 [신분]이고, [받는 사람]에게 [목적]에 대한 이메일을 [몇 자] 정도로 [톤] 써줘."

# 학습
"나는 [수준]이야. [개념]을 일상생활 비유 1개와 흔한 오해 1개를 포함해서 설명해줘."

# 비교
"[A]와 [B]를 [기준 5가지]로 비교한 표를 만들고, 상황별 추천을 알려줘."

# 브레인스토밍
"[상황]에 대한 아이디어 10개 필요해. 평범한 5개 + 엉뚱한 5개로 줘."

바이브 코딩 - 자주 쓰는 프롬프트

# 코드 설명 요청
"이 코드의 각 줄이 무슨 일을 하는지 한국어 주석으로 설명해줘.
비전공자가 가장 헷갈릴 부분 3개도 짚어줘."

# 에러 디버깅
"다음 코드를 실행했는데 에러가 났어:
[코드]
에러: [에러 메시지]
원하던 것: [의도]
원인과 해결 방법을 알려줘."

# 웹페이지 생성
"[누구]를 위한 [어떤] 웹페이지를 만들어줘.
포함 섹션: [목록]
디자인 톤: [톤]
반응형 필수. HTML+CSS 한 파일로."

# 기능 추가
"이 페이지에 [기능]을 추가해줘.
기존 구조는 건드리지 말고, 새 코드만 표시해줘."

자주 묻는 질문

Q. Claude Pro 구독은 꼭 필요한가요?

네. 무료 플랜은 사용 한도가 빠르게 차서 강의 실습에 부족합니다. Pro($20/월) 또는 학생 단체 할인을 활용해 주세요.

Q. Cursor는 유료 플랜이 필요한가요?

아닙니다. 무료 플랜으로도 한 달에 충분한 양을 쓸 수 있고, 본인이 가진 Claude Pro 구독을 연결해서 쓰면 한도가 훨씬 늘어납니다.

Q. 컴맹인데 따라갈 수 있을까요?

네. 이 강의는 "비전공자 초보"가 기본 대상입니다. AI 도구가 어려운 부분을 대신 해주기 때문에, 옛날 코딩 강의보다 진입 장벽이 훨씬 낮습니다.

Q. 결과물은 어떻게 활용할 수 있나요?

본인이 만든 페이지는 본인 소유입니다. 포트폴리오·이력서·SNS 어디든 자유롭게 활용 가능합니다.

Q. 강의 후에도 계속 배우려면?

강의 5일차 마지막에 본인 관심사에 따른 다음 단계 트랙을 안내합니다 (심화 웹개발 / AI API / 모바일 / 데이터 분석).
