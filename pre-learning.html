체크리스트 한눈에

본인이 수강하는 과정에 맞춰 체크하세요. 두 과정 모두 듣는다면 전부 필요합니다.

오전반 (AI 기초) 필수
  • 노트북 (브라우저 동작 가능한 모든 기기)
  • Claude 계정 + Pro 구독
  • Canva 계정 (무료 가능)
  • Gamma 계정 (무료 가능)
  • 구글 계정 (위 3개 가입에 사용)
오후반 (바이브 코딩) 필수
  • 노트북 (RAM 8GB 이상 권장)
  • Claude 계정 + Pro 구독
  • Cursor 에디터 설치 (무료)
  • Python 3.10+ 설치
  • GitHub 계정 (무료, 배포용)

1. 노트북 준비

최소 사양

  • OS: macOS · Windows 10/11 · Linux 모두 가능
  • RAM: 4GB 이상 (오전반) / 8GB 권장 (오후반 Cursor 사용)
  • 저장공간: 여유 10GB 이상
  • 인터넷: 안정적인 Wi-Fi (강의실 제공)
  • 웹 브라우저: Chrome / Edge / Safari 최신 버전
💡 노트북이 없다면
한라대 실습실 PC도 사용 가능합니다. 다만 본인 노트북이면 강의 후에도 본인이 만든 결과물에 계속 작업할 수 있어 권장.

2. Claude 계정 + Pro 구독 (두 과정 공통)

가입 (5분)

  1. claude.ai 접속
  2. 구글 계정 또는 이메일로 가입
  3. 한국 가능, 휴대폰 인증 필요

Pro 구독 (월 $20, 학생 단체 할인 별도 안내)

  1. 로그인 → 우측 상단 프로필 → Subscription
  2. Upgrade to Pro 클릭 → 카드 정보 입력
  3. 3개월(약 6만원)이 강의 기간 권장
⚠️ 안내
Pro 구독 없이는 강의 실습 한도가 빠르게 차서 따라가기 어렵습니다. 한라대 단체 할인 안내가 있을 수 있으니 강사 공지 확인.

가입 후 한 번 만져보기

강의 첫날 부드럽게 시작하려면 지금 한 번 채팅을 던져 보세요.

안녕, Claude. 나는 한라대학교 비전공자야.
다음 주에 AI 기초 강의를 들어. 시작하기 전에
내가 알면 좋을 AI 활용 팁 5가지를 알려줘.

3. Cursor 설치 (오후반만, 15분)

  1. cursor.sh 접속 → Download
  2. 본인 OS(macOS/Windows/Linux) 맞는 버전 다운로드
  3. 설치 후 첫 실행 → 본인 이메일로 Sign Up (무료 플랜 충분)
  4. 설정 (⚙️) → Models → claude-opus 또는 claude-sonnet 기본으로

설치 확인

  1. Cursor 실행 → File → New File → hello.py
  2. 다음 입력 후 저장:
print("Cursor 정상 작동")
  1. 오른쪽 상단 ▶️ Run 또는 터미널 python hello.py
  2. 출력 보이면 OK
⚠️ Python 없을 때
Mac: 보통 python3로 설치돼 있음. Windows: Microsoft Store에서 "Python 3.11" 무료 설치.

4. Canva 계정 (오전반만)

  1. canva.com 접속 → 구글 계정으로 가입
  2. 한국 가능, 무료 플랜으로 충분
  3. 가입 후 한 번 "디자인 만들기 → 인스타 게시물" 시도해 보기
  4. 좌측 메뉴에서 앱 → Magic Media가 보이는지 확인 (AI 이미지 생성 도구)

5. Gamma 계정 (오전반만)

  1. gamma.app 접속 → 구글 계정으로 가입
  2. 한국어 지원, 무료 플랜으로 시작
  3. 가입 후 "+ 새로 만들기 → 프레젠테이션 → AI로 생성"이 보이는지 확인

6. GitHub 계정 (오후반만)

강의 4일차에 본인 웹사이트를 인터넷에 공개할 때 사용합니다.

  1. github.com 접속 → Sign up
  2. 학교 이메일로 가입 (.ac.kr 도메인은 Student Pack 혜택 가능)
  3. 사용자명은 본인 영문이름 권장 (URL에 노출됨, 추후 변경 어려움)
  4. 가입 후 Settings → Emails에서 이메일 인증 완료
💡 GitHub Student Pack
학교 이메일로 가입했다면 education.github.com/pack에서 학생 인증 → 다양한 개발 도구 무료 (Copilot, JetBrains, Notion 등).

7. 마인드셋 점검

강의 첫날부터 부드럽게 출발하려면, 다음 세 가지 마음가짐을 갖고 오시면 됩니다.

① "외우려 하지 말기"

AI 도구의 모든 기능을 외울 필요 없습니다. "필요한 일을 묘사하는 능력"이 새 시대의 실력입니다.

② "처음 결과가 별로여도 정상"

AI 결과의 90%는 "프롬프트를 다시 다듬으면" 좋아집니다. 3번 다듬기를 기본으로 생각하세요.

③ "막혀도 강사·동료에게 즉시 물어보기"

비전공자가 가장 흔히 하는 실수는 혼자 30분 헤매는 것입니다. 막히면 3분 안에 손드세요.

💬 사전 예열 활동 (선택)
강의 전에 Claude에게 다음을 시도해 보세요. 도구 감각이 자연스럽게 잡힙니다.
  • "내 전공 분야에 AI가 어떤 식으로 활용되고 있는지 사례 5개"
  • "내가 자주 하는 [업무·과제]를 AI로 어떻게 자동화할 수 있나"
  • "비전공자가 1주일 안에 만들 수 있는 작은 AI 프로젝트 추천"

첫날 가져올 것

  • ✅ 노트북 + 충전기 (강의실 콘센트 제한될 수 있음)
  • ✅ 이어폰 (영상 시청 시 사용)
  • ✅ 본인 신분증 (수강생 확인용)
  • ✅ 필기도구 (메모 습관 권장)
  • ✅ 마음의 준비 — 5일 동안 새로운 것을 매일 시도하기

자주 묻는 사전 질문

Q. Claude Pro가 꼭 필요한가요?

네. 무료 플랜은 하루에 던질 수 있는 메시지 수가 매우 적어서, 강의 실습 따라가기에 부족합니다. 5일짜리 부트캠프 동안 1개월 구독($20)으로도 충분합니다.

Q. 코딩을 한 번도 안 해봤는데 오후반 따라갈 수 있나요?

네. 오후반은 "AI가 코드를 만들어 주는 것을 사람이 다듬는" 방식이라, 전통 코딩 강의보다 진입 장벽이 훨씬 낮습니다. 컴퓨터 사용에 거부감만 없으면 따라갈 수 있도록 설계됐습니다.

Q. 두 과정(오전+오후) 같이 들어도 되나요?

물리적으로는 가능합니다 (09:00-18:00 같은 주). 다만 1주에 40시간 학습은 부담이 큽니다. 본인 체력·집중력에 맞춰 결정하세요.

Q. 노트북 사양이 안 좋아요. 실습실 PC를 써도 되나요?

네. 다만 본인이 만든 결과물을 강의 후에도 이어가려면 본인 노트북 권장. 임시로 실습실 사용 → 결과물을 GitHub/Google Drive에 백업하는 방법도 있음.

Q. 사전 가입에 실패했어요!

강의 첫날 1교시에 강사가 함께 셋업을 진행합니다. 너무 걱정 마시고 오세요. 다만 1시간 더 미리 준비하면 학습 시간을 절약할 수 있어 추천합니다.
