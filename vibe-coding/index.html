바이브 코딩 (45H · 9일 집중)

"프로그래밍 언어를 외운다"는 시대는 끝났습니다.
Claude와 Cursor에게 자연어로 말해서, 본인의 웹사이트를 직접 만드는 9일 집중 입문 트랙입니다.

📅 6/29(월) ~ 7/9(목) · 월~금 ⏰ 5~9교시 / 5H 일 👥 정원 40명 🎯 결과물 · 개인 웹사이트

바이브 코딩이 뭐예요?

Andrej Karpathy (전 OpenAI 공동창업자, 테슬라 AI 디렉터)가 2025년 2월에 처음 쓴 표현으로, "AI에게 자연어로 말해서 코드를 만들고 즉시 실행하는 새로운 개발 스타일"을 가리킵니다.

옛날 방식

언어 문법 외우기 → 1년 → 책 따라하기 → 6개월 → 작은 프로그램 → 1년 → 본인 프로젝트

총 2-3년

바이브 코딩

Claude·Cursor 설치 → 본인 아이디어 자연어로 요청 → AI가 코드 생성 → 실행하며 다듬기

총 9일 · 45H

⚡ 핵심 마인드셋
문법을 모르는 게 부끄러운 게 아닙니다. "내가 원하는 결과를 정확히 묘사하는 능력"이 새로운 코딩 실력입니다.
어찌 보면 AI 기초 실습에서 배운 프롬프트 능력이 바로 새 시대의 프로그래밍입니다.

학습 목표

이 트랙을 마치면 다음을 할 수 있어야 합니다.

💻

개발 환경 운용

Claude + Cursor 설치 → 프로젝트 생성 → 코드 작성 → 실행까지 막힘없이 한다.

📖

코드 읽기

AI가 만들어준 코드를 줄 단위로 이해하고, 어디를 어떻게 바꿔야 할지 판단한다.

🌐

웹페이지 제작

HTML/CSS/JS로 작동하는 개인 웹사이트를 직접 만들고 인터넷에 배포한다.

🐛

오류 해결

에러 메시지를 AI에게 보여주고 문제를 진단·수정하는 디버깅 사이클을 익힌다.
