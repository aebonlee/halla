바이브 코딩이란?

2025년 2월, OpenAI 공동창업자였던 Andrej Karpathy가 자신의 트위터에 이렇게 썼습니다.

📜 원문
"There's a new kind of coding I call 'vibe coding', where you fully give in to the vibes, embrace exponentials, and forget that the code even exists."

번역: "내가 '바이브 코딩'이라고 부르는 새로운 코딩이 있다.
그저 흐름에 몸을 맡기고, AI의 폭발적 능력을 받아들이고, 코드가 존재한다는 사실조차 잊어버리는 코딩이다."

전통 코딩 vs 바이브 코딩

전통 코딩
  • 언어 문법을 외운다
  • 한 줄 한 줄 직접 친다
  • 구글에서 답을 찾는다
  • 스택오버플로 검색
  • 에러 메시지를 직접 해석
바이브 코딩
  • 원하는 결과를 묘사한다
  • AI가 코드를 만든다
  • AI에게 직접 묻는다
  • 대화로 문제 해결
  • 에러를 AI에게 보여주고 고친다

그럼 코딩을 안 배워도 되나요?

배워야 합니다.
하지만 "외우는 게 아니라 읽는 능력"이 중요해졌습니다.
AI가 만든 코드가 무슨 일을 하는지 알아야 수정도 하고, 잘못된 부분도 찾을 수 있기 때문입니다.

💡 새로운 코딩 실력의 정의
  1. 원하는 것을 정확히 묘사하는 능력 (= 좋은 프롬프트)
  2. AI가 만든 코드를 읽고 이해하는 능력 (= 독해)
  3. 안 되는 것을 알아채고 다시 요청하는 능력 (= 디버깅 마인드)

환경 세팅 - 30분 안에 끝내기

① Claude 가입 + Pro 구독 확인

  1. claude.ai 접속 → 구글 또는 이메일로 가입
  2. 오른쪽 상단 프로필 → "Subscription" 클릭
  3. "Claude Pro" 또는 "Team" 플랜이 활성화돼 있는지 확인
  4. 구독이 안 되어 있다면 강사에게 단체 구독 안내 받기

② Cursor 에디터 설치

  1. cursor.sh 접속 → "Download" 클릭 → 본인 OS 맞게 다운
  2. 설치 후 첫 실행 → 언어 한국어 선택 가능 (영어 권장)
  3. "Sign in" → 본인 이메일로 가입 (무료 플랜으로 충분)
  4. 설정 (⚙️) → "Models" → "claude-opus" 또는 "claude-sonnet"을 기본 모델로 선택
⚠️ Windows 사용자 주의
Cursor 설치 후 "Microsoft Defender SmartScreen" 경고가 뜨면 → "추가 정보" → "실행"을 클릭하세요.
정상 동작입니다.

③ 작업 폴더 만들기

  1. 바탕화면에 halla-vibe 폴더 생성
  2. Cursor 실행 → "File" → "Open Folder" → 방금 만든 폴더 선택
  3. 좌측 사이드바에 폴더가 보이면 성공

④ 첫 파이썬 파일

  1. 좌측 사이드바에서 "새 파일" 아이콘 클릭
  2. 파일 이름: hello.py
  3. 다음 코드 작성:
print("안녕, 한라대학교!")
print("나는 오늘부터 바이브 코더야.")
  1. 저장 (Ctrl/Cmd + S)
  2. 오른쪽 위 "▶️ Run" 버튼 또는 터미널에서 python hello.py
  3. 결과가 출력되면 환경 세팅 완료!
🐍 파이썬이 없어요!
"python: command not found" 에러가 뜬다면 → 강사에게 도움 요청.
Mac은 보통 python3를 사용하고, Windows는 Microsoft Store에서 Python 3.11 설치 권장.

첫 번째 바이브 코딩 - 계산기 만들기

이제 진짜 바이브 코딩을 해 봅시다.
Cursor의 AI 채팅 (Ctrl/Cmd + L)을 엽니다.

실습 ① 자연어로 요청하기

채팅창에 다음을 그대로 붙여 보세요:

파이썬으로 간단한 계산기를 만들어줘.
- 사용자에게 두 숫자를 입력받기
- 사칙연산 중 어떤 걸 할지 선택받기 (+, -, *, /)
- 결과를 보기 좋게 출력
- 0으로 나누는 경우는 에러 처리
- 한국어 안내 문구

Cursor가 코드를 만들어 줍니다.
결과 코드 예시:

def calculator():
    print("=== 계산기 ===")
    a = float(input("첫 번째 숫자: "))
    b = float(input("두 번째 숫자: "))
    op = input("연산자 (+, -, *, /): ")

    if op == "+":
        result = a + b
    elif op == "-":
        result = a - b
    elif op == "*":
        result = a * b
    elif op == "/":
        if b == 0:
            print("0으로 나눌 수 없습니다!")
            return
        result = a / b
    else:
        print("잘못된 연산자입니다.")
        return

    print(f"결과: {a} {op} {b} = {result}")

calculator()

실습 ② 코드 적용 + 실행

  1. AI가 만든 코드 위에 "Apply" 버튼이 보입니다 → 클릭하면 자동으로 파일에 들어감
  2. 파일 이름은 calc.py 정도로 저장
  3. 실행해 보고 정상 작동 확인

실습 ③ 자연어로 수정 요청

같은 채팅에서 이어서 요청합니다.

  • "한 번 계산하고 끝내지 말고, 사용자가 '종료'라고 입력할 때까지 반복하게 해줘."
  • "계산 결과를 history.txt 파일에 기록해줘."
  • "%(나머지)와 **(거듭제곱) 연산도 추가해줘."
🔁 이것이 바이브 코딩의 핵심 사이클
자연어 요청 → AI 코드 생성 → 실행 → 결과 보기 → 자연어로 수정 → 반복
이 사이클이 빠르게 돌수록 좋은 바이브 코더입니다.

Cursor 핵심 단축키 5가지

단축키 (Mac / Win) 기능
Cmd/Ctrl + LAI 채팅 열기
Cmd/Ctrl + K인라인 코드 생성/수정 (선택한 부분만)
Cmd/Ctrl + IComposer (여러 파일 한꺼번에 수정)
TabAI 자동 완성 수락
Cmd/Ctrl + \에디터 화면 분할 (코드 + AI 채팅 동시 보기)

오늘의 과제

  1. 오늘 만든 계산기에 본인이 직접 기능 1가지 더 추가하기 (예: 평균 계산, 절댓값 등)
  2. 내일 사용할 Python 파일 1개를 미리 만들어 오기 - "주사위 게임" 또는 "가위바위보" 게임.
