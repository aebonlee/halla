텍스트 / 대화

가장 기본이 되는 생성형 AI. 글쓰기·요약·번역·아이디어·학습 도우미·코드까지 거의 모든 일에 쓰입니다.

🟧

Claude

Anthropic. 긴 문서 처리·코드·정확성 우위. 본 강의 메인 도구.
🟢

ChatGPT

OpenAI. 이미지·웹검색·플러그인 등 부가 기능 풍부.
한국어 보통.
🔷

Gemini

Google. Workspace(Gmail·Docs)와 통합 강함.
한국어 자연스러움.
🔍

Perplexity

출처 인용 검색에 특화. "AI 위키" 같은 정확성 우선 쿼리에.
💡 도구 선택 기준
  • 학습·코드·긴 문서 → Claude
  • 이미지 보기·웹 검색 → ChatGPT 또는 Gemini
  • 출처 필요한 사실 조사 → Perplexity
  • 구글 메일·문서 자동화 → Gemini

이미지 생성

"글로 묘사 → 그림" 패턴.
포스터·일러스트·로고·SNS 콘텐츠에 활용.

🎨

Canva AI (Magic Media)

비전공자에게 가장 친숙.
템플릿·디자인 통합. 본 강의 메인.
🎭

Midjourney

예술적 품질 최상.
Discord 기반.
유료.
🔵

Bing Image Creator

DALL-E 3 기반 무료.
빠른 시도용.
🧪

Stable Diffusion

오픈소스.
로컬 실행 가능, 세밀한 제어.

좋은 이미지 프롬프트 공식

[주제] + [구도] + [스타일] + [분위기/색감] + [디테일 수준]

예시:
"한라산 정상에서 일출을 바라보는 등산객의 뒷모습,
시네마틱 라이팅, 황금시간대, 4K 디테일"

발표 준비 시간을 1/5로.

조합 예시 — 도구를 합치면 강해집니다

① 인스타 카드뉴스 1편 (30분)

  1. Claude → 주제 잡고 카피 5장 분량 작성
  2. Canva AI Magic Media → 배경 이미지 5장 생성
  3. Canva 캐러셀 템플릿에 텍스트·이미지 배치
  4. Claude → 인스타 캡션 + 해시태그 20개 자동 생성

② 발표용 영상 자료 (1시간)

  1. Claude → 발표 개요 잡기
  2. Gamma → 슬라이드 10장 자동 생성
  3. ElevenLabs → 슬라이드 내레이션 음성
  4. Canva Magic Video → 슬라이드 + 음성 영상화

③ 신곡 1곡 (15분)

  1. Claude → 가사 작성 (테마·운율 지정)
  2. Suno → 가사 + 장르(K-Pop) 입력 → 완성 곡
  3. Canva AI → 앨범 커버 이미지

④ 본인 웹사이트 (1일, 본 강의 오후반)

  1. Cursor + Claude → HTML/CSS/JS 자연어로 생성
  2. Canva AI → 프로필·헤더 이미지 생성
  3. Netlify/Vercel → 무료 배포 (10분)

AI 활용 윤리 — 알아둬야 할 5가지

  1. 출처와 사실 확인 — AI는 "그럴듯한 거짓말"을 만들 때가 있음.
    인용·통계·이름은 반드시 검증.
  2. 저작권 — Canva·Gamma·Claude 결과물은 일반 상업 사용 가능하지만 각 플랫폼 약관 확인.
    실명·실제 브랜드 이미지는 가급적 피하기.
  3. AI 사용 표기 — 학교 과제·논문에는 AI 사용 비중과 방법을 솔직히 밝히기.
    한라대학교 정책 확인 필수.
  4. 개인정보 — AI 채팅창에 본인·타인의 민감 정보(주민번호·신용카드·비밀번호) 입력 금지.
  5. 의사결정 책임 — AI는 도구.
    최종 판단과 책임은 본인. "AI가 그렇게 답했다"는 핑계가 안 됨.
