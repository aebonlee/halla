Gradio란

파이썬 몇 줄로 웹 데모 UI를 만드는 도구입니다. 만든 LangChain 체인을 누구나 쓰게 공개할 때 가장 빠릅니다.

!pip install gradio

최소 예제

import gradio as gr

def greet(name):
    return f"안녕하세요, {name}님!"

# fn=함수, inputs/outputs=컴포넌트
gr.Interface(fn=greet, inputs="text", outputs="text").launch()

챗봇 — ChatInterface

챗봇은 ChatInterface 한 줄이면 됩니다. fn(message, history) 형태로 함수만 주면 끝.

import gradio as gr

def respond(message, history):
    # history: [(user, bot), ...] 이전 대화
    return f"'{message}' 라고 하셨네요!"

gr.ChatInterface(respond, title="내 챗봇").launch()

Blocks 레이아웃

자유롭게 화면을 구성하려면 gr.Blocks를 씁니다.

import gradio as gr

with gr.Blocks() as demo:
    gr.Markdown("## 내 데모")
    chatbot = gr.Chatbot()
    msg = gr.Textbox(placeholder="입력...")
    def respond(message, chat_history):
        chat_history.append((message, "답: " + message))
        return "", chat_history
    msg.submit(respond, [msg, chatbot], [msg, chatbot])

demo.launch()

LangChain 연결

응답 함수 안에서 우리가 만든 체인을 invoke하면 됩니다.

# rag, tutor 등 미리 만든 LangChain 체인을 그대로 연결
def respond(message, history):
    return rag.invoke(message)      # 체인 호출 결과를 그대로 반환

gr.ChatInterface(respond, title="문서 Q&A 봇").launch(share=True)

스트리밍 출력

한 글자씩 나오게 하려면 yield로 부분 응답을 흘려보냅니다.

def respond(message, history):
    partial = ""
    for chunk in llm.stream(message):   # LangChain stream
        partial += chunk.content
        yield partial                   # 누적해서 계속 반환

gr.ChatInterface(respond).launch()

공개 링크 share=True

demo.launch(share=True)   # 72시간 동안 유효한 공유 URL 생성
🔐 키 주의
공유 링크엔 누구나 접속합니다. API 키는 환경변수로만 두고, 비용 상한을 걸어두세요.

실전 팁 & 실수

  • queue() — 동시 접속이 많으면 demo.queue().launch()
  • 예시 버튼 — gr.ChatInterface(fn, examples=["안녕","요약해줘"])
  • 느린 응답 → 스트리밍 또는 gpt-4o-mini 사용
  • Colab에서 share=True가 막히면 런타임 재시작 후 재실행
  • 키 노출 금지 — 화면 공유 전 키 출력 셀 삭제
