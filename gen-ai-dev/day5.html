학습 목표

  • 내 PDF를 근거로 답하는 RAG 챗봇을 처음부터 끝까지 만들 수 있다.
  • 만든 챗봇을 Gradio/LangServe로 배포해 URL로 공유할 수 있다.

PDF 문서 적재

from langchain_community.document_loaders import PyPDFLoader, PyPDFDirectoryLoader
from langchain_text_splitters import CharacterTextSplitter
from langchain_openai import OpenAIEmbeddings
from langchain_chroma import Chroma

docs = PyPDFDirectoryLoader("my_pdfs/").load()
chunks = CharacterTextSplitter(chunk_size=1000, chunk_overlap=100).split_documents(docs)
db = Chroma.from_documents(chunks, OpenAIEmbeddings())
retriever = db.as_retriever(search_kwargs={"k": 4})

RAG 체인 완성

검색한 근거(context)와 질문을 함께 프롬프트에 넣어 답을 만듭니다 — 오픈북 시험처럼.

from langchain_core.prompts import ChatPromptTemplate
from langchain_core.runnables import RunnablePassthrough
from langchain_core.output_parsers import StrOutputParser
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template(
  "아래 context만 근거로 답해. 모르면 모른다고 해.\n\ncontext: {context}\n\n질문: {question}")

rag = ({"context": retriever, "question": RunnablePassthrough()}
       | prompt | ChatOpenAI(model="gpt-4o-mini") | StrOutputParser())
print(rag.invoke("이 문서의 핵심 3가지는?"))

근거 표시 · 환각 방지

  • "context에 없으면 모른다고 답하라"는 가드레일을 프롬프트에 명시
  • 답과 함께 근거가 된 청크를 보여줘 신뢰도 확인

배포 — Gradio · LangServe

완성한 체인에 화면을 붙이거나 API로 공개해 "서비스"로 만듭니다.

import gradio as gr
def chat(msg, history): return rag.invoke(msg)
gr.ChatInterface(chat).launch(share=True)   # 공유 가능한 URL 생성
🔐 배포 시 키
공개 배포에는 API 키가 노출되지 않도록 서버/환경변수로 관리합니다.
