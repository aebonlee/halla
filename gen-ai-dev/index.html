생성형 AI 개발 (48H)

"AI를 쓰는 사람"에서 "AI를 만드는 사람"으로. 프롬프트를 넘어 API·챗봇·RAG·에이전트까지, 6일 종일 집중으로 나만의 생성형 AI 서비스를 직접 완성합니다.

📅 6/29(월) ~ 7/4(토) ⏰ 1~3교시 · 5~9교시 (종일 8H) 👥 누구나 (혼합) 📍 한라대학교 본관

이 과정이 특별한 이유

오전반·오후반의 다음 단계 — "직접 만드는" 데 초점을 맞춘 종일 집중 과정입니다.

🚀

만드는 사람으로

도구 사용을 넘어 API를 호출하고 앱을 조립하는, "만드는 쪽"의 감각을 익힙니다.

🧩

누구나 시작

선수 지식이 없어도 됩니다. 노코드·로우코드 도구로 시작해 점진적으로 코드를 다룹니다.

🗄️

내 데이터로

RAG로 내 문서를 학습한 AI, 이미지·음성 멀티모달까지 — 나만의 데이터로 똑똑해지는 AI.

🛠️

배포까지

만든 결과물을 인터넷에 올려 누구나 쓸 수 있는 진짜 "서비스" 형태로 마무리합니다.

학습 목표

이 과정을 마치면 다음을 할 수 있어야 합니다.

🔑

API 다루기

Claude API 키를 발급받아 프로그램에서 생성형 AI를 호출하고 응답을 다룰 수 있다.

💬

챗봇 제작

역할·맥락이 담긴 시스템 프롬프트로 목적에 맞는 대화형 AI 앱을 만들 수 있다.

📚

RAG 구성

내 문서를 근거로 답하는 검색 기반 AI(RAG)의 원리를 이해하고 구성할 수 있다.

📢

서비스 발표

기획부터 배포까지 거친 본인의 생성형 AI 서비스를 시연하며 발표할 수 있다.

6일 커리큘럼

각 일차를 클릭하면 상세 강의 노트를 볼 수 있습니다. (상세 노트는 등록 수강생에게 열립니다)

일차날짜주제시간
1 6/29 (월)
생성형 AI 개발 입문 + 환경 세팅
생성형 AI 개발의 지형 / Claude·Cursor 세팅 / API 키 발급 / 노코드 도구 둘러보기
 8H 2 6/30 (화)
프롬프트 엔지니어링 & API 기초
시스템 프롬프트 설계 / Claude API 호출 / 파라미터(temperature·max_tokens) / 구조화 출력
 8H 3 7/1 (수)
나만의 AI 챗봇 만들기
대화 흐름 설계 / 페르소나 챗봇 / 노코드+로우코드 조합 / 화면 붙이기
 8H 4 7/2 (목)
내 데이터로 똑똑한 AI — RAG & 멀티모달
RAG 원리 / 내 문서 기반 Q&A / 이미지 생성·분석 / 멀티모달 활용
 8H 5 7/3 (금)
AI 에이전트 & 자동화 + 배포
도구 사용(tool use) / 작업 자동화 워크플로 / 웹에 배포해 서비스로 공개
 8H 6 7/4 (토)
미니 프로젝트 ③ — 기획·구현·발표
나만의 생성형 AI 서비스 완성 / 시연 발표(1인 1서비스) / 피어 피드백
 8H

하루 운영 (종일 8H)

오전 1~3교시 + 오후 5~9교시. 점심 이후 실습 비중이 커집니다.

오전
1~3교시 · 개념 + 데모
그날 주제의 개념과 동작 원리, 강사 라이브 데모
 오후
5~9교시 · 실습 + 빌드
직접 만들기 중심. 1:1 코칭, 결과물 점진적 완성
