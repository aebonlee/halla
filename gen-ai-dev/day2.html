학습 목표

  • 좋은 프롬프트의 원칙을 적용하고 템플릿으로 재사용할 수 있다.
  • LCEL " | "로 prompt·model·parser를 연결할 수 있다.
  • RunnablePassthrough/assign으로 체인 입력을 유연하게 다룰 수 있다.
  • 커스텀 OutputParser로 모델 출력을 가공하고 외부 함수와 연결할 수 있다.

프롬프트 작성 원칙

  • 명확성·구체성 — 무엇을, 어떤 형식으로 원하는지 분명히
  • 맥락 제공 — 배경 정보를 주면 환각이 줄고 관련성↑
  • 간결함 — 핵심에 집중, 불필요한 정보 배제
  • 예시(few-shot) — 원하는 결과의 예를 1~2개 제시

프롬프트 템플릿 심화

단일 프롬프트는 PromptTemplate, 대화형은 ChatPromptTemplate. 템플릿끼리 +로 조합도 됩니다.

from langchain_core.prompts import PromptTemplate
tmpl = PromptTemplate.from_template("{name}님은 {age}살입니다. {language}로 인사해줘")
print(tmpl.format(name="홍길동", age=20, language="영어"))

표현언어(LCEL) — 파이프

LangChain의 정수는 " | " 파이프로 구성요소를 잇는 것입니다. 데이터가 왼쪽→오른쪽으로 흐릅니다.

from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template("Tell me a short joke about {topic}")
chain  = prompt | ChatOpenAI() | StrOutputParser()
print(chain.invoke({"topic": "ice cream"}))
💡 StrOutputParser
모델 응답에서 문자열만 깔끔히 추출. 체인 끝에 붙이면 .content 없이 바로 문자열.

LCEL은 어떻게 동작하나

" | "는 마법이 아니라 파이썬 __or__ 연산자입니다. 함수를 이어 붙이는 작은 클래스를 직접 만들어 원리를 체험합니다.

class Runnable:
    def __init__(self, func): self.func = func
    def __or__(self, other):              # a | b
        def chained(*a, **k): return other(self.func(*a, **k))
        return Runnable(chained)
    def __call__(self, *a, **k): return self.func(*a, **k)

add_five     = Runnable(lambda x: x + 5)
multiply_two = Runnable(lambda x: x * 2)
chain = add_five | multiply_two
print(chain(3))   # (3+5)*2 = 16

Runnable로 입력 다루기

입력을 그대로 넘기거나(Passthrough), 일부 값을 함수로 채워(assign) 체인을 유연하게 만듭니다.

from langchain_core.runnables import RunnablePassthrough

template = "Please generate three dialogue sentences in {language} on {topic} for a {level} level"
prompt = ChatPromptTemplate.from_template(template)

# topic 하나만 넣으면 language/level은 함수가 자동으로 채움
chain = (RunnablePassthrough.assign(
            language=lambda _: "English",
            level=lambda _: "beginner")
         | prompt | ChatOpenAI() | StrOutputParser())
print(chain.invoke({"topic": "travel"}))

커스텀 출력 파서

모델 답에서 원하는 부분만 뽑도록 파서를 직접 만들 수 있습니다.

from langchain_core.output_parsers import BaseOutputParser

class CodeOutputParser(BaseOutputParser):
    def parse(self, text: str):
        # ```text\n15**0.3432\n``` → "15**0.3432"
        return text.strip().split("```")[1].split("\n")[1]

수학 체인 (실전)

자연어 질문 → LLM이 수식 생성 → 파서로 수식 추출 → numexpr로 실제 계산. 여러 블록을 이어 "하는 일"을 만듭니다.

import numexpr
from langchain_core.runnables import RunnablePassthrough

system = ("You make ONLY an argument for numexpr.evaluate.\n"
          "Respond as a ```text``` code block only.")
prompt = ChatPromptTemplate.from_messages([("system", system), ("human", "{user_input}")])

chain = ({"user_input": RunnablePassthrough()}
         | prompt | ChatOpenAI(temperature=0) | CodeOutputParser() | numexpr.evaluate)
print(chain.invoke("What is 15 raised to the .3432 power?"))

오늘의 실습 미션

  1. few-shot 예시를 넣어 일정한 형식의 답을 받는 체인
  2. LCEL prompt | model | StrOutputParser() 체인 1개 완성
  3. 수학 체인을 변형해 "단위 변환기" 같은 나만의 도구 만들기

과제

  1. RunnablePassthrough.assign으로 변수 1개만 받아 동작하는 체인 만들기
