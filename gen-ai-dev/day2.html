2일차 · 프롬프트 & 표현언어(LCEL)

학습 목표

  • 명확·구체·맥락 있는 프롬프트 작성 원칙을 적용할 수 있다.
  • PromptTemplate / ChatPromptTemplate / few-shot을 사용할 수 있다.
  • LCEL " | "로 prompt·model·parser를 연결하고 Runnable로 입력을 다룰 수 있다.

프롬프트 작성 원칙

  • 명확성·구체성 — 무엇을, 어떤 형식으로 원하는지 분명히
  • 맥락 제공 — 배경 정보를 주면 환각이 줄고 관련성이 높아짐
  • 간결함 — 핵심에 집중, 불필요한 정보 배제
  • 예시(few-shot) — 원하는 결과의 예를 1~2개 보여주기

프롬프트 템플릿

from langchain_core.prompts import PromptTemplate

tmpl = PromptTemplate.from_template("{name}님은 {age}살입니다. {language}로 인사해줘")
print(tmpl.format(name="홍길동", age=20, language="영어"))

대화형은 ChatPromptTemplate.from_messages([...]) 로 system/human 메시지를 함께 구성합니다.

표현언어(LCEL) — 핵심

LangChain의 정수는 " | " 파이프로 구성요소를 잇는 것입니다. 데이터가 왼쪽 → 오른쪽으로 흐릅니다.

from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template("{topic}에 대한 짧은 농담 하나")
chain  = prompt | ChatOpenAI() | StrOutputParser()
print(chain.invoke({"topic": "아이스크림"}))
💡 StrOutputParser
모델 응답 객체에서 문자열만 깔끔히 뽑아줍니다. 체인 끝에 붙이면 .content 없이 바로 문자열.

Runnable 조합

입력을 그대로 넘기거나(Passthrough), 병렬 실행(Parallel), 함수 끼우기(Lambda)로 흐름을 유연하게 만듭니다.

from langchain_core.runnables import RunnablePassthrough

chain = {"topic": RunnablePassthrough()} | prompt | ChatOpenAI() | StrOutputParser()
print(chain.invoke("커피"))   # 문자열 하나만 넣어도 동작

커스텀 출력 파서

모델 답에서 원하는 부분만 추출하도록 파서를 직접 만들 수 있습니다 (예: 수학 체인에서 코드만 뽑기).

from langchain_core.output_parsers import BaseOutputParser

class CodeOutputParser(BaseOutputParser):
    def parse(self, text):
        return text.split("```")[1] if "```" in text else text

chain = chat_prompt | ChatOpenAI(temperature=0) | CodeOutputParser()
