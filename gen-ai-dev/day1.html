1일차 · 개발 입문 + 환경 세팅

생성형 AI 개발의 전체 지형을 잡고, Claude·Cursor 개발 환경과 API 키까지 한 번에 세팅합니다.

오늘의 시간표

오전 · 1~3교시
개념 + 라이브 데모 — 생성형 AI 개발이 무엇이고 어디까지 가능한지, 도구 지형 한눈에
점심
점심 시간
오후 · 5~9교시
실습 + 빌드 — Claude·Cursor 설치/로그인, API 키 발급, 첫 호출 확인

학습 목표

  • "AI를 쓰는 것"과 "AI로 만드는 것"의 차이를 설명할 수 있다.
  • 생성형 AI 개발에 쓰이는 도구 지형(노코드 → 로우코드 → 코드)을 구분할 수 있다.
  • Claude·Cursor 개발 환경을 세팅하고 API 키를 발급받아 첫 호출을 확인할 수 있다.

생성형 AI 개발의 지형

이 과정은 "프롬프트를 잘 쓰는 사람"에서 한 걸음 더 나아가, AI를 부품처럼 끼워 넣어 무언가를 만드는 사람이 되는 6일입니다.

세 가지 층위

  • 노코드 — 코드 없이 화면 클릭으로 AI 앱 조립 (예: 챗봇 빌더, 자동화 도구)
  • 로우코드 — 약간의 코드 + 도구 조합. 가장 가성비 좋은 구간
  • 코드 — API를 직접 호출. 자유도가 가장 높음
💡 이 과정의 전략
노코드로 빠르게 감을 잡고, 필요한 곳에만 코드를 더합니다. "전부 코딩"이 아니라 "필요한 만큼"이 핵심입니다.

6일 동안 만들 것

  • Day 2~3: 목적이 분명한 나만의 챗봇
  • Day 4: 내 문서로 답하는 RAG Q&A
  • Day 5: 스스로 도구를 쓰는 에이전트 + 웹 배포
  • Day 6: 이 모두를 합친 나만의 생성형 AI 서비스

개발 환경 세팅

  1. Claude — claude.ai 가입 (이 과정 필수, 3개월 구독 권장)
  2. Cursor — cursor.sh에서 설치. AI 내장 코드 에디터
  3. 계정 준비 — Google 계정(로그인·배포용), GitHub 계정(코드 보관·배포용)
⚠️ 미리 해두면 좋아요
설치/가입은 시간이 걸립니다. 가능하면 사전학습 페이지를 보고 1일차 전에 준비해 오세요.

API 키 발급 + 첫 호출

API 키는 "프로그램이 AI에게 말 걸 수 있는 비밀번호"입니다.

  1. console.anthropic.com 접속 → 로그인
  2. API Keys → Create Key → 키 복사 (한 번만 보이니 안전한 곳에 보관)
  3. Cursor에서 첫 호출 테스트 — 강사가 실습 코드를 함께 실행합니다
🔐 키 보안 — 매우 중요
API 키는 절대 공개 코드(깃허브 등)에 올리지 않습니다. 노출되면 요금 폭탄·오남용 위험이 있습니다. 키는 환경변수/비공개 설정에만 둡니다.
💰 비용 감각
API는 사용한 토큰만큼 과금됩니다. 실습 수준에서는 매우 적은 금액이지만, "토큰 = 돈"이라는 감각을 처음부터 가집니다.

오늘의 과제

  1. Claude·Cursor 설치 완료 스크린샷 남기기
  2. API 키 발급 후 안전하게 보관 (노출 금지 규칙 지키기)
  3. "내가 만들고 싶은 생성형 AI" 아이디어 3개를 메모해 오기 — Day 6 프로젝트 씨앗
