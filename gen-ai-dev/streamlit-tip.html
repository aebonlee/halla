Streamlit이란

데이터·AI 앱을 파이썬 스크립트로 만드는 프레임워크입니다. Gradio보다 자유도가 높아 대시보드형 앱에 좋습니다.

!pip install streamlit

설치 & 실행

Gradio와 달리 .py 파일로 만들고 명령어로 실행합니다.

# app.py 로 저장 후 터미널에서:
streamlit run app.py
# (Colab은 localtunnel/ngrok 필요 — 로컬·클라우드 권장)

위젯·출력

import streamlit as st

st.title("내 AI 앱")
st.write("일반 텍스트/표/차트 출력")

name = st.text_input("이름")          # 입력창
if st.button("인사"):                 # 버튼
    st.success(f"{name}님 안녕하세요")  # 결과 박스

세션 상태

Streamlit은 입력 때마다 스크립트를 처음부터 다시 실행합니다. 값을 유지하려면 st.session_state를 씁니다 (대화 기록의 핵심).

import streamlit as st
if "messages" not in st.session_state:
    st.session_state.messages = []      # 최초 1회 초기화
st.session_state.messages.append({"role": "user", "content": "안녕"})

챗 UI

import streamlit as st

if "messages" not in st.session_state:
    st.session_state.messages = []

# 이전 대화 출력
for m in st.session_state.messages:
    with st.chat_message(m["role"]):
        st.write(m["content"])

# 입력 받기
if prompt := st.chat_input("메시지를 입력하세요"):
    st.session_state.messages.append({"role": "user", "content": prompt})
    answer = "답변: " + prompt          # ← 여기에 LangChain 체인 연결
    st.session_state.messages.append({"role": "assistant", "content": answer})
    st.rerun()

캐싱 (벡터스토어)

벡터스토어 같은 무거운 객체는 매 실행마다 다시 만들면 느립니다. @st.cache_resource로 한 번만 만듭니다.

import streamlit as st

@st.cache_resource          # 한 번 만들고 재사용
def load_db():
    from langchain_chroma import Chroma
    from langchain_openai import OpenAIEmbeddings
    return Chroma(persist_directory="./db", embedding_function=OpenAIEmbeddings())

db = load_db()

LangChain 연결

answer = rag.invoke(prompt)          # 체인 결과를
st.session_state.messages.append({"role": "assistant", "content": answer})

배포 & secrets

Streamlit Community Cloud에 무료 배포할 수 있습니다. API 키는 st.secrets로 안전하게 관리합니다.

import streamlit as st, os
os.environ["OPENAI_API_KEY"] = st.secrets["OPENAI_API_KEY"]   # 비밀값
# 배포 시 대시보드 Settings → Secrets 에 키 등록
🔐 키 주의
키를 코드에 직접 쓰지 말고 st.secrets/환경변수로. GitHub에 키 커밋 금지.

실전 팁 & 실수

  • "왜 변수가 초기화되지?" → Streamlit은 매 입력마다 재실행. 유지할 값은 session_state
  • 무거운 로딩 → @st.cache_resource(객체)·@st.cache_data(데이터)
  • st.rerun()으로 화면 갱신
  • 레이아웃 → st.columns, st.sidebar
  • 대시보드형(차트·표)엔 Streamlit, 초간단 데모엔 Gradio
