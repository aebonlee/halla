파이썬 A to Z

이 과정에 필요한 최소 파이썬 — 변수·자료구조·반복문·함수·컴프리헨션과 LangChain 패턴.

이 페이지에서
  • 왜 파이썬인가
  • 변수와 자료형
  • 자료구조
  • 조건문·반복문
  • 함수
  • 모듈·패키지·pip
  • 파일·예외
  • 문자열·f-string
  • 컴프리헨션
  • LangChain에서 자주 쓰는 패턴
  • 자주 하는 실수

왜 파이썬인가

생성형 AI 개발의 사실상 표준 언어입니다. 이 과정에선 "코드를 다 외우기"보다 "읽고 고치기"가 목표 — 아래는 그때 필요한 최소 파이썬입니다.

📝 사용법
막히면 이 페이지에서 문법을 찾고, AI에게 "이 코드 설명해줘"라고 물으세요.

변수와 자료형

name = "한라"      # 문자열(str)
age  = 20           # 정수(int)
pi   = 3.14         # 실수(float)
ok   = True         # 불(bool): True / False
none = None         # 값 없음

print(type(name), type(age), type(pi), type(ok))
print(f"{name}대학교, {age}살")   # f-string 으로 값 끼워넣기

자료구조

# 리스트(list): 순서O, 변경O
nums = [1, 2, 3];  nums.append(4);  print(nums[0], nums[-1])

# 딕셔너리(dict): 키-값 (LangChain invoke 입력이 dict!)
person = {"name": "홍길동", "age": 20};  print(person["name"])

# 튜플(tuple): 순서O, 변경X
point = (3, 4)

# 집합(set): 중복X
tags = {"ai", "ai", "ml"};  print(tags)   # {'ai','ml'}
💡 중요
LangChain의 chain.invoke({"topic": "AI"})처럼 dict를 가장 많이 씁니다.

조건문·반복문

# 조건문
score = 85
if score >= 90:   grade = "A"
elif score >= 80: grade = "B"
else:             grade = "C"

# for 반복
for i in range(3):        # 0,1,2
    print(i)
for q in ["a", "b"]:      # 리스트 순회
    print(q)

# while 반복 (챗봇 루프에서 사용)
while True:
    msg = input("입력(quit=종료): ")
    if msg == "quit": break

함수

def greet(name, lang="한국어"):     # lang은 기본값
    """이름을 받아 인사말을 돌려준다."""
    return f"{name}님 안녕하세요 ({lang})"

print(greet("철수"))            # 철수님 안녕하세요 (한국어)
print(greet("Tom", "English"))  # 인자 바꿔 호출

# 람다(lambda): 한 줄 함수 — LCEL에서 RunnableLambda로 자주 씀
double = lambda x: x * 2
print(double(5))

파일·예외

# 파일 읽기 (with: 자동으로 닫아줌)
with open("notes.txt", "r", encoding="utf-8") as f:
    text = f.read()

# 예외 처리 (에러로 멈추지 않게)
try:
    n = int("abc")           # 에러 발생
except ValueError as e:
    print("숫자가 아니에요:", e)

문자열·f-string

s = "  LangChain  "
print(s.strip())            # 양끝 공백 제거 → "LangChain"
print(s.lower())            # 소문자
print("a,b,c".split(","))   # ['a','b','c']
print("\n".join(["x","y"])) # 줄바꿈으로 합치기 (format_docs에서 사용)

topic, n = "AI", 3
print(f"{topic} 관련 문장 {n}개 만들어줘")   # f-string

컴프리헨션

# 리스트 컴프리헨션: for를 한 줄로 (RAG format_docs에서 자주 사용)
squares = [x*x for x in range(5)]          # [0,1,4,9,16]
texts   = [d.upper() for d in ["a","b"]]   # ['A','B']

# 조건 포함
evens = [x for x in range(10) if x % 2 == 0]

LangChain에서 자주 쓰는 패턴

# 1) dict 입력
chain.invoke({"topic": "AI"})

# 2) 람다로 함수 끼우기
{"context": lambda q: format_docs(retriever.invoke(q)), "question": lambda q: q}

# 3) join + 컴프리헨션 (검색 문서 합치기)
def format_docs(docs):
    return "\n\n".join(d.page_content for d in docs)

# 4) with 로 파일/리소스 안전하게
📝 이 4가지
dict · lambda · join+컴프리헨션 · with — LangChain 코드의 8할이 이 패턴입니다.

자주 하는 실수

  • 들여쓰기(indent) 안 맞음 → 파이썬은 들여쓰기로 블록을 구분 (스페이스 4칸 통일)
  • =(대입)와 ==(비교) 혼동
  • 딕셔너리 키 오타 → KeyError
  • 문자열/정수 섞기 → "3" + 3는 에러 (int("3")+3)
  • Colab에서 설치는 !pip, 코드에선 import
