3일차 · 분류·감성분석 & 대화형 챗봇

학습 목표

  • LLM 체인으로 텍스트 분류(감성분석)를 수행하고 결과를 정량 평가할 수 있다.
  • 대화 이력을 유지하는 챗봇을 LangChain으로 만들 수 있다.

감성분석 분류 체인

리뷰/댓글을 긍정(1)·부정(0)으로 분류하는 체인을 만듭니다. 출력 파서로 숫자만 받아 프로그램이 처리할 수 있게 합니다.

from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "다음 리뷰가 긍정이면 1, 부정이면 0만 출력해."),
    ("human", "{input_text}"),
])
chain = prompt | ChatOpenAI(temperature=0) | classificationOutputParser()

def langchain_llm(text):
    return chain.invoke({"input_text": text})   # 1 또는 0

분류 성능 평가

데이터를 불러와 예측하고, 정확도·혼동행렬로 모델이 얼마나 맞는지 측정합니다.

import pandas as pd
from sklearn.metrics import accuracy_score, confusion_matrix

preds = [langchain_llm(t) for t in df["review"]]
print("정확도:", accuracy_score(df["label"], preds))
print(confusion_matrix(df["label"], preds))
📝 감각
"AI가 잘하는지"를 느낌이 아니라 숫자로 확인하는 습관 — 실무의 기본입니다.

대화형 챗봇

system 프롬프트로 페르소나를 잡고, MessagesPlaceholder로 이전 대화를 끼워 넣습니다.

from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

prompt = ChatPromptTemplate.from_messages([
    ("system", "너는 친절한 학습 도우미야."),
    MessagesPlaceholder(variable_name="history"),
    ("human", "{input}"),
])
chain = prompt | ChatOpenAI(model="gpt-4o-mini", temperature=0.1)

대화 기억 유지

이전 user/ai 메시지를 저장했다 매번 함께 보내면 맥락을 이어갑니다.

from langchain_core.chat_history import InMemoryChatMessageHistory

history = InMemoryChatMessageHistory()
# "그거 칼로리는?" 처럼 앞 대화를 가리키는 질문도 이해
💡 길어지면
대화가 길수록 토큰(비용)이 늘어납니다. 오래된 대화는 요약해 넣는 기법을 소개합니다.
