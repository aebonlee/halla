학습 목표

  • LLM 체인으로 텍스트 분류(감성분석)를 수행하고 결과를 정량 평가할 수 있다.
  • 대화 이력을 유지하는 챗봇을 LangChain으로 만들 수 있다.
  • 세션별로 대화 기록을 분리·관리할 수 있다.

감성분석 분류 체인

리뷰/댓글을 긍정(1)·부정(0)으로 분류합니다. system 프롬프트로 "숫자만 출력"하도록 강하게 지시합니다.

from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "Act as a sentiment classifier. Output '1' if positive, '0' if negative. Only the integer."),
    ("human", "{input_text}"),
])
chat = ChatOpenAI(temperature=0)

정수 출력 파서

모델이 "1" 같은 문자열을 주면, 파서로 정수로 바꿔 이후 계산(정확도 등)에 쓸 수 있게 합니다.

from langchain_core.output_parsers import BaseOutputParser

class ClassificationOutputParser(BaseOutputParser):
    def parse(self, text: str):
        return 1 if "1" in text else 0

chain = prompt | chat | ClassificationOutputParser()
def classify(t): return chain.invoke({"input_text": t})

데이터 로드 & 일괄 분류

실제 댓글 데이터셋을 내려받아 pandas로 읽고, 행마다 분류 체인을 돌립니다.

import urllib.request, pandas as pd
urllib.request.urlretrieve(
  "https://raw.githubusercontent.com/tykimos/tykimos.github.io/master/warehouse/dataset/tarr_train.txt",
  "tarr_train.txt")
df = pd.read_csv("tarr_train.txt", sep="\t")
preds  = [classify(c) for c in df["comment"]]
actual = list(df["label"])

정확도·혼동행렬 평가

"느낌"이 아니라 숫자로 성능을 확인합니다.

from sklearn.metrics import accuracy_score, confusion_matrix
print("정확도:", accuracy_score(actual, preds))
print(confusion_matrix(actual, preds, labels=[1, 0]))
📝 실무 기본
분류기·요약기 등은 항상 정량 평가와 함께. 혼동행렬로 "어떤 걸 자주 틀리는지"까지 봅니다.

대화형 챗봇 구조

system 프롬프트로 페르소나를 잡고, MessagesPlaceholder로 이전 대화를 끼워 넣습니다.

from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_core.output_parsers import StrOutputParser

prompt = ChatPromptTemplate.from_messages([
    ("system", "당신은 친절한 음식점 사장입니다. 리뷰에 정중히 답하세요."),
    MessagesPlaceholder(variable_name="chat_history"),
    ("human", "{input_text}"),
])
chain = prompt | ChatOpenAI(model="gpt-4o-mini", temperature=0.1) | StrOutputParser()

대화 기억 유지

이전 user/ai 메시지를 저장했다 매번 함께 보내면 맥락을 이어갑니다. 길어지면 최근 N개만 사용합니다.

from langchain_core.chat_history import InMemoryChatMessageHistory
history = InMemoryChatMessageHistory()
def load_memory(_): return history.messages[-6:]   # 최근 3쌍

세션별 기록 관리

여러 사용자를 다루려면 RunnableWithMessageHistory로 session_id별 기록을 분리합니다.

from langchain_core.runnables.history import RunnableWithMessageHistory
store = {}
def get_session_history(sid):
    store.setdefault(sid, InMemoryChatMessageHistory())
    return store[sid]

bot = RunnableWithMessageHistory(chain, get_session_history,
        input_messages_key="input_text", history_messages_key="chat_history")
bot.invoke({"input_text":"음식이 너무 짰어요"},
           config={"configurable":{"session_id":"user-1"}})

오늘의 실습 미션

  1. 내 주제의 분류 체인(예: 문의 유형/스팸 여부) + 정확도 측정
  2. 페르소나가 분명한 챗봇(예: 친구·튜터·사장) 만들기
  3. 맥락을 3턴 이상 이어가는 대화 시연

과제

  2. 세션 2개로 각각 다른 대화가 유지되는지 확인
